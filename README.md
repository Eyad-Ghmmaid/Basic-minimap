# Basic minimap
Projektbeschreibung

Projekt: Basic Minimap

Kurzbeschreibung:
Dieses Unity-Projekt enthält eine einfache, modulare Minimap-Implementierung für 3D- oder 2D-Szenen. Die Minimap verwendet eine separate Kamera (oder ein RenderTexture) und UI-Elemente, um die Position des Spielers und relevanter Objekte in Echtzeit anzuzeigen.

Hauptfeatures:
- Minimap-Kamera mit eigener Layer-Filterung
- Anzeige von Spieler-Position und wichtigen Objekten
- Einfache Integration über Prefab(s) und Inspector-Referenzen

Technische Hinweise:
- Zielplattform: Unity (siehe verwendete Unity-Version im Projekt). Die C#-Skripte sind kompatibel mit `.NET 4.x` API Compatibility Level (Projekt verwendet .NET Framework 4.7.1 Zielsetzung für Bibliotheken).
- Asset-Ordner: Suche nach Prefabs unter `Assets/Prefabs` oder `Assets/UI`.

Weitere Informationen:
- Siehe `QuickStart.md` im Projektstamm für eine kurze Anleitung zum Öffnen des Projekts, Einfügen der Minimap in eine Szene und Fehlerbehebung.
